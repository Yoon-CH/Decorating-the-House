## 🎊 과제 구현 사항 및 영상
- 제공받은 사진, 가구정보 API를 연동하여 레이아웃 구현
- 돋보기 모양 클릭 시 X아이콘 생성 및 가구정보 생성
- 동시에 하단 제품 리스트 border 색상 변경으로 선택 표시
